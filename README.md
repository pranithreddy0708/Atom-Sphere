🧪 Atom Simulation (Interactive Web Project)
📌 Project Overview

This project is an interactive atom simulation built using HTML, CSS, and JavaScript. It visually represents atomic structures including protons, neutrons, and electrons, allowing users to explore different elements dynamically.

The simulation was developed as part of a project presented at a Two-Day Inter-College Project Expo (VISHESH-2024).

🚀 Features
🎯 Interactive atom selection (Hydrogen → Oganesson)
⚛️ Visual representation of:
Protons
Neutrons
Electrons
🔄 Real-time animation of particle movement
🌀 Electron orbit visualization (shell-based)
🎨 Dynamic color rendering for particles
📊 Supports up to 118 elements
🛠️ Technologies Used
HTML5 – Structure of the web page
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Logic and animations
Canvas API – Rendering the atom simulation

Open the application in your browser
Use the dropdown menu (top-right corner)
Select any element (e.g., Oxygen, Carbon, Gold)
Watch the atom structure animate in real-time
🧠 Working Concept
The nucleus contains protons and neutrons that move slightly to simulate vibration.
Electrons revolve in circular orbits (shells).

Electron configuration follows basic shell limits:

2, 8, 18, 32, ...
Animation is achieved using requestAnimationFrame().
